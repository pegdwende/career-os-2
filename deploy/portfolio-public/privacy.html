Privacy

Public portfolio privacy notes.

This site is designed to share public professional information while avoiding private recruiter notes, interview prep, target-role scoring, internal architecture details, and local file paths.

What is public

The portfolio, public resume page, one-page resume PDF, skill evidence matrix, and architecture judgment examples are public. Anything shown on those pages can be viewed, copied, indexed, or saved by visitors and search engines.

AI chat

The AI assistant answers from a public profile knowledge file only. It is instructed not to answer from private recruiter notes, interview prep notes, missing information checklists, target-role match scoring, salary assumptions, local file paths, hidden metadata, or confidential employer details.

Chat requests are rate-limited by network address before any AI request is made. Raw chat transcript storage is optional and happens only when the visitor checks the transcript storage consent box.

Contact form

Recruiter contact details are stored only when the contact consent box is checked. The form includes a hidden spam-trap field to reject common automated bot submissions.

Resume downloads

Resume downloads are served through a rate-limited endpoint. Download events store a salted IP hash for abuse prevention and aggregate counting; raw IP addresses are not intentionally stored by the application for resume downloads.

Retention

  • Consented chat transcripts: up to 90 days.
  • Consented contact submissions: up to 90 days.
  • Resume download event records: up to 90 days.
  • Rate-limit counters: short-lived windows based on the configured limit period.

Deployment notes

OpenAI and Redis keys must be stored only as Vercel environment variables. They should not be committed to the repository or exposed in browser JavaScript.
